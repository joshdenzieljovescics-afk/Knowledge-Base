# Workload Agent - System Architecture Documentation

## 1. Overview

The **Workload Agent** is a specialized microservice for workforce planning and capacity analysis. It processes time study data, calculates standard times, and provides workload projections based on volume inputs. The agent is designed to work with VFP-Productivity_TMS.xlsx format files.

| Property | Value |
|----------|-------|
| **Service Name** | Workload Agent |
| **Default Port** | 8008 |
| **Version** | 1.0.0 |
| **Technology** | FastAPI (Python) |
| **Primary Function** | Time Study Analysis & Workforce Planning |

## 9. Recommendation Engine

The Workload Agent generates contextual recommendations based on the analysis:

```python
def generate_recommendations(projection: WorkloadProjection) -> List[Recommendation]:
    recommendations = []
    
    # Staffing recommendations
    if projection.fte_gap > 0.5:
        recommendations.append(Recommendation(
            type="staffing",
            message=f"Consider adding {ceil(projection.fte_gap)} FTE to meet demand",
            priority="high" if projection.fte_gap > 2 else "medium"
        ))
    elif projection.fte_gap < -1:
        recommendations.append(Recommendation(
            type="staffing",
            message=f"Team may be overstaffed by {abs(projection.fte_gap):.1f} FTE",
            priority="medium"
        ))
    
    # Utilization recommendations
    if projection.utilization_percent > 95:
        recommendations.append(Recommendation(
            type="capacity",
            message="Utilization above 95% - high risk of delays and burnout",
            priority="critical"
        ))
    elif projection.utilization_percent > 90:
        recommendations.append(Recommendation(
            type="capacity",
            message="Utilization above 90% - limited flexibility for peak demand",
            priority="high"
        ))
    elif projection.utilization_percent < 70:
        recommendations.append(Recommendation(
            type="capacity",
            message="Utilization below 70% - opportunity for cross-training",
            priority="low"
        ))
    
    return recommendations
```

## 14. Glossary

| Term | Definition |
|------|------------|
| **Observed Time (OT)** | Raw time measured for a task |
| **Performance Rating Factor (PRF)** | Adjustment for worker pace relative to "normal" |
| **Normal Time (NT)** | Time for a task at normal pace (OT Ã— PRF) |
| **Allowance** | Additional time for fatigue, personal needs, delays |
| **Standard Time (ST)** | Time to complete task including allowances |
| **Productivity** | Output rate (units per hour) |
| **FTE** | Full-Time Equivalent (1 FTE = one full-time worker) |
| **Utilization** | Percentage of available time actually used |
